---
description: Workflow conventions
---

- Documentation changes require format, lint, and link validation. Run `pnpm check:docs` to verify docs before committing.
- Translation memory for i18n: docs/.i18n/zh-CN.tm.jsonl (generated); update English docs → adjust glossary (docs/.i18n/glossary.zh-CN.json) → run scripts/docs-i18n → apply targeted fixes only if instructed; see docs/.i18n/README.md
