- Chat commands in WhatsApp/Telegram/Slack/Google Chat/Microsoft Teams/WebChat: /status (session status), /new or /reset (reset session), /compact (compact context via summary), /think &lt;level&gt; (off|minimal|low|medium|high|xhigh for GPT-5.2+Codex only), /verbose on|off, /usage off|tokens|full (per-response usage footer), /restart (owner-only in groups), /activation mention|always (group activation toggle, groups only).
